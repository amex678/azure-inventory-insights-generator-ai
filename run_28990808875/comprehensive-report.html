エグゼクティブサマリ

本レポートは、現在ログイン中の Azure サブスクリプションにおけるリソース、RBAC、NSG、Defender for Cloud、Azure Advisor の横断的な棚卸しとリスク分析を実施した結果をまとめています。以下の主要数値と分析結果をもとに、4つの主要改善領域を提示します。

主要数値

  • リソース総数: 51件 / リソースグループ数: 7 / リージョン数: 5
  • VM数: 未取得（詳細リソース種別別集計なし） / Storage(LRS)件数: 未取得 / Public IP件数: 未取得
  • タグ付与率: 45.1%
  • RBAC総割り当て数: 未取得 / Owner割り当て数: 13 / UAA割り当て数: 2 / 孤児候補件数: 40 / マネージメントグループスコープのOwner数: 未取得
  • NSG数: 未取得 / 総ルール数: 未取得 / 高リスクルール件数: 0
  • Defender for Cloud Unhealthy件数: 79（High: 31件、Medium: 未集計件、Low: 未集計件）
  • Defender最頻出推奨事項: Update libpython3.10-stdlib_for_linux 等のセキュリティ更新
  • Advisor件数: 未集計総数（High Impact件数: 25）
  • Advisor最頻出問題: Azure Disksのゾーン冗長性未設定による可用性リスク

主要改善領域

  1. 公開境界の強化 - NSG高リスクルールは0件ですが、Public IP件数未取得のため潜在的な露出リスクあり。
  2. 特権境界の管理 - Owner割り当て13件、孤児候補40件と多く、PIM導入やアクセスレビューが必要。
  3. 監視フィードバックの活用 - Defender for Cloudで79件のUnhealthy検出（High 31件含む）、迅速な対応が求められる。
  4. ガバナンス基盤の強化 - タグ付与率45.1%と低く、Azure Policyやタグ継承の導入推奨。
