Azure 総合運用監査レポート

生成日時: 2026-07-09 03:48:55 UTC

エグゼクティブサマリ

本レポートは、現在ログイン中の Azure サブスクリプションにおけるリソース、RBAC、NSG、Defender for Cloud、Azure Advisor の横断的な棚卸しとリスク分析結果をまとめたものです。

  • リソース総数: 51件 / リソースグループ数: 7 / リージョン数: 5
  • 代表的リソース件数: VM、Storage(LRS)、Public IP は詳細未取得のため「データなし」
  • タグ付与率: 45.1%
  • RBAC 総割り当て数: 未取得（Owner: 13、UAA: 2、孤児候補: 40、マネージメントグループスコープの Owner: 未取得）
  • NSG 数: 未取得 / 総ルール数: 未取得 / 高リスクルール件数: 0
  • Defender for Cloud Unhealthy 件数: 79（High: 31、Medium: 未集計だが約20件以上、Low: 未取得）
  • Defender for Cloud 最頻出推奨事項: Linux VM の複数重要パッケージ更新（例: libpython3.10 系、vim 系など）
  • Azure Advisor 件数: 未取得総数（High Impact 件数: 25）
  • Azure Advisor 最頻出問題: ストレージアカウントのゾーン冗長化未設定および Linux VM のディスク暗号化未設定
  • Defender for Cloud は有効化済みでデータ取得可能
  • Az.Advisor は有効化済みでデータ取得可能

本サブスクリプションにおける主要な改善領域は以下の4つです。

  1. 公開境界：Public IP と高リスク NSG ルールは現在0件ですが、公開境界の管理強化が必要です。
  2. 特権境界：Owner 13件、孤児候補 40件と特権ロールの恒久付与と管理不全の懸念があります。
  3. 監視フィードバック：Defender for Cloud と Advisor の推奨事項を活用した継続的なセキュリティ改善が求められます。
  4. ガバナンス基盤：タグ付与率 45.1% と低く、ストレージの冗長性設定も不十分なため、ポリシー強化が必要です。

潜在リスク Top 5

1. 公開境界の露出 Low

本サブスクリプションの高リスク NSG ルール件数は 0件、Public IP 件数は未取得ですが、公開管理ポートの許可がないことからリスクは低いと判断されます。

推奨対策: Bastion + JIT (Just-In-Time VM Access)、Application Gateway WAF の導入を検討してください。

参考リンク: Azure Bastion / JIT VM Access / Application Gateway WAF

2. 特権ロールの恒久付与と blast radius Medium

Owner 割り当て数は 13件、孤児候補は 40件 と多く、特権ロールの恒久付与によるリスクが懸念されます。

推奨対策: PIM (Privileged Identity Management)、マネージド ID の活用、Access Review の実施を推奨します。

参考リンク: Azure AD PIM / マネージド ID / Access Reviews

3. Defender for Cloud のセキュリティ態勢ギャップ High

Defender for Cloud の Unhealthy 件数は 79件、うち High Severity は 31件 あり、Linux VM の重要パッケージ更新が多数未対応です。

推奨対策: Foundational CSPM の導入、Secure Score の月次レビューを実施してください。

参考リンク: Defender for Cloud 入門 / Secure Score / CSPM (Cloud Security Posture Management)

4. Advisor フィードバックループ不在 High

Azure Advisor の High Impact 件数は 25件 あり、ストレージのゾーン冗長化未設定や VM のディスク暗号化未設定が目立ちます。

推奨対策: Advisor 推奨事項の月次レビューと Cost Management 連動を推進してください。

参考リンク: Azure Advisor / Cost Management / Advisor ベストプラクティス
